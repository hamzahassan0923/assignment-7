Keen Keeper

📖 Short Description

Keen Keeper is a simple and user-friendly note-taking web application inspired by Google Keep. It allows users to create, manage, and organize their notes easily in a clean and responsive interface.

⚙️ Technologies Used
1.React.js
2.JavaScript (ES6)
3.CSS3
4.Netlify (for deployment)

⭐ Key Features
1.I have a section for friends where I can call or message them, and I can also make video calls.

2.There is a separate page called the timeline, where I can see all activities along with the time—like when calls and messages were made.

3.There is also a friends analytics section, a stats page, where I can see how many calls and messages I have sent, received, and it beautifully displays this data in graphs.
